\doxysubsection{Detailed Description}
Type for arrays of functions to be registered by lua\+L\+\_\+setfuncs. 

name is the function name and func is a pointer to the function. Any array of \doxylink{structlua_l___reg}{lua\+L\+\_\+\+Reg} must end with a sentinel entry in which both name and func are NULL.
