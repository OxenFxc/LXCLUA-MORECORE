lu\+\_\+byte lua\+O\+\_\+codeparam (\begin{DoxyParamCaption}\item[{unsigned int}]{p}{}\end{DoxyParamCaption})}



Encodes \textquotesingle{}p\textquotesingle{}\% as a floating-\/point byte (eeeexxxx). 

The exponent is represented using excess-\/7. Mimicking IEEE 754, the representation normalizes the number when possible, assuming an extra 1 before the mantissa (xxxx) and adding one to the exponent (eeee) to signal that. So, the real value is (1xxxx) \texorpdfstring{$\ast$}{*} 2\texorpdfstring{$^\wedge$}{\string^}(eeee -\/ 7 -\/ 1) if eeee != 0, and (xxxx) \texorpdfstring{$\ast$}{*} 2\texorpdfstring{$^\wedge$}{\string^}-\/7 otherwise (subnormal numbers).

\end{DoxyParams}
